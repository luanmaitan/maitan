# Ícones

Coloque os ícones SVG das redes sociais aqui.

## Ícones Necessários

Para a página `/sobre` funcionar completamente, você precisa adicionar:

- `github.svg` - Ícone do GitHub
- `linkedin.svg` - Ícone do LinkedIn  
- `instagram.svg` - Ícone do Instagram
- `substack.svg` - Ícone do Substack

## Onde Encontrar

Você pode baixar os ícones oficiais de:
- [Simple Icons](https://simpleicons.org/) - Ícones SVG de marcas
- [Heroicons](https://heroicons.com/) - Ícones gerais
- Sites oficiais das redes sociais (geralmente na seção de recursos/brand)

## Formato

Os ícones devem ser arquivos SVG. Exemplo de estrutura:

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="currentColor">
  <!-- conteúdo do ícone -->
</svg>
```

## Tamanho Recomendado

Os ícones serão redimensionados automaticamente, mas recomenda-se usar viewBox padrão como `0 0 24 24` para melhor compatibilidade.
